# MkDocs

## Starting mkdocs
* install python and pip
```pip install mkdocs```
```pip install mkdocs-cinder```
```mkdocs serve --dev-addr=0.0.0.0:9000```

## Building
```mkdocs build -d {output}```

## Help
```mkdocs build --help```
